### Module 1: os
- **Purpose**: Interact with the operating system.
- **Functionality**: File and directory manipulation.
- **Example Usage**:
```python
import os
os.listdir('.')
```

### Module 2: sys
- **Purpose**: Access system-specific parameters and functions.
- **Functionality**: Interpret command-line arguments.
- **Example Usage**:
